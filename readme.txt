== Installation ==

This section describes how to install the plugin and get it working.

1. Download: Grab the plugin zip file from the GitHub repository or using this link.
2. Upload: Navigate to your WordPress admin panel, head to Plugins > Add New, click “Upload Plugin,” and select the downloaded zip file.
3. Activate: Once uploaded, activate the plugin from the Plugins page.
